# 모듈: Event Calendar

## 역할

연준, 지표 발표, 정부 정책, 산업/기업 이벤트를 구조화해 market/sector/stock 판단에 연결한다.

## 현재 상태

상태: Planned.

## 예정 입력

- FOMC 일정과 발표.
- CPI, PCE, 고용, GDP, ISM 등 주요 지표.
- 미국/한국 정부 정책.
- 산업/기업 이벤트.
- 실적 발표와 가이던스.

## 예정 출력

- 이벤트별 중요도.
- 적용 시장/섹터/종목.
- 예상 영향 방향.
- 이벤트 전후 체크아웃 지표.
- 관련 verdict invalidation 조건.

## 다음 개선

- 수동 event ledger schema.
- event type taxonomy.
- event-driven watchlist.
