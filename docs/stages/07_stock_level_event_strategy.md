# 7단계: 개별주 이벤트/상황 기반 매수 전략 구체화

## 목표

개별 종목별 이벤트, 차트, 리스크를 고려해 진입 타점과 손절 조건을 구체화한다.

## 주요 분석 요소

- 종목별 주요 이벤트.
- 체크아웃 지표.
- 지지선과 저항선.
- 피보나치 확장.
- 볼린저밴드.
- 차트 패턴.
- RSI.
- 진입 타점과 스탑로스.

## 주요 출력

- 종목별 매수 조건.
- 종목별 손절 조건.
- 이벤트 확인 방법.
- 전략 무효화 조건.

## 현재 구현 상태

상태: Planned.

현재 price structure와 relative strength 일부만 존재한다. 이벤트 기반 체크아웃 지표와 차트 기반 rulebook은 아직 없다.

## 다음 구현

1. chart indicator 모듈 후보를 정의한다.
2. 진입/손절을 “추천”이 아니라 rule-based plan으로 표현한다.
3. 종목별 이벤트 ledger와 연결한다.
