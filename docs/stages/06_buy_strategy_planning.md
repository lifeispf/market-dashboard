# 6단계: 매수 전략 수립

## 목표

현재 보유 주식 상황과 목표 포트폴리오를 비교해 리밸런싱 계획과 매수 전략을 수립한다.

## 주요 입력

- 현재 보유 종목, 수량, 평균단가.
- 목표 포트폴리오 비중.
- 종목별 action hint.
- 시장/섹터 리스크.
- 계좌 현금과 주문 가능 금액.

## 주요 출력

- 리밸런싱 필요 금액.
- 매수/매도 후보.
- 주문 우선순위.
- 분할 매수 계획.
- 실행 전 확인할 체크리스트.

## 현재 구현 상태

상태: Planned.

현재 보유 포트폴리오를 입력으로 받아 리밸런싱 계획을 생성하는 모듈은 없다.

## 다음 구현

1. portfolio holdings input schema를 만든다.
2. 목표 비중과 현재 비중의 차이를 계산하는 순수 함수를 만든다.
3. 주문 실행 전 단계의 plan만 출력한다.
